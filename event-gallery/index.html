Event 1

Language: en Complexity: Intermediate

Polymer is a first of its kind library. It’s designed to leverage the emerging Web Component standards, to make web app development a breeze! If you’re new to Web Components, or curious about the latest and greatest features that Polymer has to offer, we want to see you in this session! We’ll kick off with an overview of Polymer to get everyone up to speed, explore the growing component ecosystem, and take a look at the next generation of web apps. Then we’ll dive into building out elements and apps with your choice of self-guided code labs. Grab your thinking caps: #itshackademic with Polymer!

Event Description

Language: en Complexity: Expert

We’ve created a series of codelabs to take you from Polymer beginner to expert element wrangler in no time. We’ll start with a quick, interactive run through of Polymer and the features that it offers. Then we’ll step it up a notch and build our first full fledged mobile app in Polymer. Finally, we’ll explore what it means to work with an API like Google Maps, using nothing but HTML. If you’re curious about the future of web app development, come join us for some serious hacking fun.

Event 3

Language: en Complexity: Expert

Ecmascript 6 is the future version of Javascript and is packed with goodies and our life much more convenient and web programming much more fun. It is only a few months away - so let's start preparing now. In this talk, you will get an overview of the new features, including modules, classes, promises, arrow functions, generators, template strings. You will also see how you can start using ES6 today with the Google's Traceur compiler.

Angular or Backbone: Go Mobile!

Language: en Complexity: Intermediate

Angular or Backbone, which one you will use in your mobile app? How could you develop a mobile app across iOS, Android or windows devices? This talk will take an intimate look at two of today’s most popular frameworks, Angular and Backbone and explore their differences. We’ll show how Apache Cordova opens the world of mobile app development to web developers. In the session, we will demonstrate a “To Do” app using Angular and Backbone, with access to native device capabilities. We’ll compare the frameworks when transported to the world of mobile app development.

Mobile Growth Hacking: Lessons Learned from Growing from Zero to 100M Installs

Language: en Complexity: Intermediate

The old adage of the ‘field of dreams’ isn’t true – create a mobile app, publish it and the installs will just happen. That is no longer true anymore. You spent all this time, effort, money and energy building an app. Why is no one downloading it? You would expect that if you build an awesome feature-rich application, it would kill it in the App Store. Growth hacking techniques that drive installs is an art. Building a mobile application is only the first hurdle in developing a successful application. There is a lot of intricacy that goes into growth hacking and gaining not only users, but loyal users and driving real DAUs. Come to this session and learn all the insiders growth hacking tips from Dolphin Browser.
